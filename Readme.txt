Overview
GDP Trends Visualization is a Python-based project that analyzes and visualizes GDP data for different countries over time. It provides an interactive way to explore economic trends using clear and insightful charts. The project combines data cleaning, statistical representation, and interactive controls for a better analytical experience.

Features
Load and preprocess GDP data from CSV files.

Remove unnecessary columns for focused analysis.

Generate clear and professional GDP trend charts.

Interactive selection of countries and time periods using widgets.

Simple workflow suitable for students, analysts, and researchers.
